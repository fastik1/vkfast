# Установка

## Требования

* PHP ^8.0
* Composer последней версии

{% hint style="info" %}
Подробнее в файле composer.json
{% endhint %}

## Установка

Выполняется через утилиту composer, вводить в командную строку при глобальной установке composer'а:

```
composer require fastik1/vkfast
```
